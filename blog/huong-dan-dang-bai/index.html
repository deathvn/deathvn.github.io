Hướng dẫn viết bài

Tất cả các bài đăng đều được viết bằng Markdown. Bạn có thể dùng Notepad (hoặc bất kì Code Editor nào) để edit trực tiếp file Markdown, nhưng để tiện cho việc preview và sửa lỗi, bạn nên cài đặt Moeditor.

Hình 1. Giao diện của Moeditor

Bước 1. Tạo File Markdown và Folder chứa ảnh.

  • Tạo một thư mục đặt tên là tiêu đề ngắn gọn của bài viết, tên thư mục viết không dấu và cách nhau bằng dấu - Ví dụ: blog-thu-nghiem
  • Tạo một file đặt cùng tên với thư mục vừa tạo, nhưng có thêm phần mở rộng .md Ví dụ: blog-thu-nghiem.md

Hình 2. Cấu trúc thư mục có được sau khi tạo

Bước 2. Viết bài vào File Markdown vừa tạo.

Mở file .md vừa tạo và thêm vào đoạn sau:

---
layout: post
title:  "<Tiêu đề bài viết>"
author: <Nickname tác giả, dưới 20 kí tự, viết thường, không dấu>
categories: [ <thể loại> ]
image: <đường dẫn đến file hình ảnh dùng làm ảnh bìa cho bài viết>
---

Ví dụ:

---
layout: post
title:  "Blog demo: Lorem Ipsum? Thử nghiệm"
author: deathvn
categories: [ demo ]
image: blog-thu-nghiem/title.jpg
---

Soạn thảo với file Markdown

Tiêu đề và đoạn văn

Dùng dấu # để biểu thị tiêu đề, và nhấn Enter (xuống dòng) 2 lần để cách giữa các đoạn văn.

Ví dụ:

# Tiêu đề 1
## Tiêu đề 1.1
### Tiêu đề 1.2
Đoạn văn 1

Đoạn văn 2

# Tiêu đề 2
Đoạn văn 1

Đoạn văn 2

# Tiêu đề 3

Hình 3. Đoạn văn và tiêu đề với Markdown

Chèn đường dẫn liên kết đến trang khác

Cú pháp:

[text](<URL>)

Ví dụ:

[Google](https://www.google.com)

Google

Chèn thêm hình ảnh

Cách 1. dùng URL của hình ảnh, (còn gọi là hotlink).

Lưu ý: Hotlink mà không được phép là hành vi phạm tội. (Bạn còn nhớ vụ scandal của nettruyen chứ???)

![mô tả ảnh](<URL>)

Ví dụ:

![random image](https://source.unsplash.com/random/800x600)
_Hình 4. Một ảnh lấy ngẫu nhiên từ unsplash_

Hình 4. Một ảnh lấy ngẫu nhiên từ unsplash

Cách 2. tải ảnh và lưu về máy.

Lưu ý: Hình ảnh không nên quá lớn, kích cỡ ảnh hợp lý là chiều ngang không vượt quá 1500px, Vì ảnh lớn hơn đều bị trình duyệt thu nhỏ lại cho vừa với màn hình

  • Lưu ảnh ở folder vừa tạo ở Bước 1.
  • Cú pháp tương tự khi chèn ảnh bằng URL, nhưng thay bằng đường dẫn tới ảnh.
![mô tả ảnh](<tên folder tạo ở Bước 1>/<tên ảnh>)

Ví dụ:

![cat](huong-dan-dang-bai/cat.jpg)
